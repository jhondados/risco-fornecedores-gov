# 🏛️ Risco de Fornecedores

> Avaliação de integridade e risco de fornecedores públicos. Cruza dados de sanções, processos e idoneidade com ML e análise de grafo.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/risco-fornecedores-gov/

## 📊 Resultados-chave (KPIs)
**890K** Fornecedores · **12K** Alto risco · **cruzadas** Sanções · **95%** Precisão

## 🧠 Tecnologias de IA
Graph ML · Risk Scoring · NLP

## ✨ O que faz
- Score de integridade
- Cruzamento CEIS/CNEP
- Análise de vínculos (grafo)
- Due diligence automática

## 🗄️ Fonte de dados real
[CEIS + CNEP + Transparência](https://portaldatransparencia.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com CEIS + CNEP + Transparência
2. **Processamento com IA** — Graph ML · Risk Scoring · NLP
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
